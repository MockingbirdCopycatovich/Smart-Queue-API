# Smart Queue & Appointment API

Backend API for managing smart queues and appointment scheduling
for service centers and clinics.

## Features
- Appointment booking with race condition protection
- Role-based access (clients, operators, admins)
- JWT authentication (access + refresh)
- Background tasks (email notifications, TTL handling)
- Dockerized environment

## Tech Stack
- FastAPI
- PostgreSQL
- Redis
- Celery
- Docker / Docker Compose
